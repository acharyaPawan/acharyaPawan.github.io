Wanted to stop juggling different copy shortcuts between macOS and Linux tmux sessions.

```
bind-key -T copy-mode-vi y send-keys -X copy-pipe-and-cancel 'pbcopy'
bind-key -T copy-mode-vi Y send-keys -X copy-line
```

I also learned that `wl-copy` on Linux can be swapped in automatically by wrapping
the `copy-pipe-and-cancel` command in a tiny shell script that checks for `pbcopy`.
Now triple copy shortcuts (editor, shell, tmux) all match muscle memory.
